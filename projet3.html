ShortCut

Site web avec algo A*

Dans le cadre d'un projet scolaire en groupe de 4, nous devions améliorer un projet existant en optimisant les performances et en adaptant l'interface.

Sujet

Le projet visait à évaluer et optimiser un code existant pour calculer des trajets entre deux villes à partir d'une base de données. Le principal défi résidait dans l'optimisation des performances, de la qualité du logiciel et de l'ergonomie de l'application. L'objectif principal était de réduire le temps de calcul initialement long en proposant des solutions adaptées. Le projet a abouti à une amélioration significative du temps de calcul, rendant l'application beaucoup plus rapide. L'interface a également été améliorée avec l'ajout d'une carte et l'utilisation de JavaScript et AJAX pour de nouvelles fonctionnalités. Ce projet a permis d'acquérir de nouvelles compétences, de mettre en pratique les connaissances du semestre, de travailler en équipe et d'appliquer la méthodologie agile.

ShortCut

Create a website with A* algorithm

As part of a group project with 4 members, we had to improve an existing project by enhancing performance and adapting the interface.

Topic

The project aimed to assess and optimize existing code for calculating routes between two cities from a database. The main challenge was optimizing performance, software quality, and application ergonomics. The primary goal was to reduce the originally long calculation time by proposing suitable solutions. The project resulted in a significant improvement in calculation time, making the application much faster. The interface was also enhanced with the addition of a map and the use of JavaScript and AJAX for new features. This project allowed us to acquire new skills, apply semester knowledge in practice, work in a team, and apply agile methodology.

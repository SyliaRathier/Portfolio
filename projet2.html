Créer un site web de Qyestions / Réponses

Dans le cadre d'un projet scolaire en groupe de 4, nous devions créer un site web permettant à des utilisateurs d'intéragir sur des sujets choisis

Sujet

    Créer un site de Questions / Réponses

  • Un utilisateur peut créer une question après avoir fait une demande à l'administrateur. Si la demande est validée, l'utilisateur peut poser une question, choisir les personnes pour y répondre et choisir les personnes qui voteront pour la meilleure réponse.
  • Les utilisateurs sélectionnés pour répondre peuvent demander de l'aide à d'autres utilisateurs pour les aider à répondre en travaillant sur la même réponse.
  • Une fois la date de fin pour répondre atteinte, un vote pour connaître la meilleure réponse est réalisé par les utilisateurs sélectionnés.
  • Une fois le vote terminé, la meilleure réponse est mise en première page du site.

Your Voice

Créer un site web de Questions / Réponses

Dans le cadre d'un projet scolaire en groupe de 4, nous devions créer un site web permettant à des utilisateurs d'interagir sur des sujets choisis.

Sujet

    Créer un site de Questions / Réponses

  • Un utilisateur peut créer une question après avoir fait une demande à l'administrateur. Si la demande est validée, l'utilisateur peut poser une question, choisir les personnes pour y répondre et choisir les personnes qui voteront pour la meilleure réponse.
  • Les utilisateurs sélectionnés pour répondre peuvent demander de l'aide à d'autres utilisateurs pour les aider à répondre en travaillant sur la même réponse.
  • Une fois la date de fin pour répondre atteinte, un vote pour connaître la meilleure réponse est réalisé par les utilisateurs sélectionnés.
  • Une fois le vote terminé, la meilleure réponse est mise en première page du site.

Your Voice

Create a Q&A Website

As part of a group project with 4 members, we had to create a website allowing users to interact on chosen topics.

Topic

    Create a Q&A website.

  • A user can create a question after making a request to the administrator. If the request is approved, the user can ask a question, choose people to answer it, and select people to vote for the best answer.
  • Selected users to answer can ask for help from other users to collaborate on the same answer.
  • Once the deadline for answering is reached, a vote to determine the best answer is conducted by the selected users.
  • After the vote is completed, the best answer is featured on the front page of the site.
